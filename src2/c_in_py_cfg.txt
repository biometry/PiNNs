# Calling preles.c from python

1: Getting source code of preles
- Github: https://github.com/MikkoPeltoniemi/Rpreles
git clone https://github.com/MikkoPeltoniemi/Rpreles

- c source code in ./src

2: to compile .c scripts 
- download/install mingw64: http://mingw-w64.org/doku.php

3: to use gcc
- add path to mingw64 e.g. 'C:/mingw64/bin' to PATH in system environment variables

4: create shared library
gcc -fPIC -shared -o call_preles.so call_preles.c gpp.c initruns.c preles.c water.c

5: in python
from ctypes import *

lib = cdll.LoadLibrary('call_preles.so')
prel = lib.call_preles
